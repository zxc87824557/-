進度條 (Progress)

  1. 使用 .progress 作為包裝器來指示進度條的最大值。
  2. 內部的 .progress-bar 來表示到目前為止的進度。
  3. .progress-bar 需要行內樣式，通用類別的 Class 或自訂 CSS 來設置它們的寬度。
  4. .progress-bar 還需要一些 role 和 aria 屬性來使其具親和性。

帶條紋的

將 .progress-bar-striped 加到任何 .progress-bar 中，透過 CSS 漸變對進度條的背景顏色加上條紋。

動態條紋

條紋漸變也可以製成動畫。 將 .progress-bar-animated 加到 .progress-bar 中，透過 CSS3 繪製從右到左的動態效果。
