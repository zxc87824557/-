Range Inputs

表單欄位會用 .form-group 區塊包住，label 的 for 屬性值對應到欄位的 id 屬性值。

readonly 與 disabled

readonly 的 value，使用者只能唯讀不能修改，get 與 post 傳值可以將值丟出去，但只能作用在 input 與 textarea 上

disabled 的 value 完全禁止修改，也不會傳值，可作用於所有表單元素。(其他元素也可以，但 a 元素不行)
